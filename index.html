Mapglyphs-China

--- The CSS Map Font\nFont of China provinces, fork me on github.com.

This project is inspired by the project Map Glyphs 1.0 by @tedgrajeda - http://mapglyphs.com - @mapglyphs.

--- The shape data is from wikipedia.

This project doesn't contain any font defined in @mapglyphs.

Usage

Step 1: Download Map China Glyphs.

You need to download the latest glyphs file.

Step 2: Unzip the glyphs file.

Make sure you can import all files. (.tff, .eot, .woff, .css)

Step 3: One one line of code needed

You need to import the css file in the zip file.

<link href="path/to/mapglyphs-china.css" rel="stylesheet">

Basic Usage

For normal usage, just use mg map-cn-icon name

<i class="mg map-cn-tw"></i> map-cn-tw
